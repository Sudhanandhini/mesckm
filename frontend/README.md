# MES College Website

A modern, responsive website for MES M.S. Padmavathamma M.K. Sambasiva Setty First Grade College For Women, built with React, Vite, and Tailwind CSS.

## Features

- ✅ Modern React 18 with Vite
- ✅ Tailwind CSS for styling
- ✅ React Router for navigation
- ✅ Responsive design
- ✅ Theme colors matching the original website
- ✅ Complete page structure with all sections

## Theme Colors

- **Primary**: #8B7355 (Brown/Olive Gold)
- **Secondary**: #6B5D4F
- **Accent**: #DC143C (Red for headings)
- **Gold**: #D4AF37

## Pages Included

1. Home - Welcome page with important announcements
2. About us - College history and principal's desk
3. Donor - Information about college donors
4. Course Regulations - Courses offered and admission details
5. Facilities - Library, laboratories, and computer lab
6. Faculty - Teaching staff details
7. Non-Teaching - Non-teaching staff details
8. Forum (Arts, Commerce, Science)
9. NSS - National Service Scheme details
10. Scholarship - Available scholarship programs
11. Rank Holders - University rank holders list
12. Yoga Wellness Centre - Yoga programs and activities
13. Contact us - Contact information and form

## Installation

1. Navigate to the project directory:
```bash
cd mes-college-website
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and visit: http://localhost:5173

## Build for Production

```bash
npm run build
```

The build files will be in the `dist` folder.
